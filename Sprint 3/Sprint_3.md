# 📊 Sprint 3 – KPI Definition & Dashboard Development

## 🎯 Objective

The objective of this sprint is to define key performance indicators (KPIs) and build an interactive dashboard to visualize intern performance insights derived from the exploratory data analysis.

---

## 📌 Key Performance Indicators (KPIs)

The following KPIs were defined to measure intern performance effectively:

* **Total Interns** – Total number of interns in the dataset
* **Total Tasks Completed** – Overall tasks completed by all interns
* **Average Performance Rate** – Average performance score across interns
* **Average Attendance** – Average attendance percentage
* **Average Productivity Ratio** – Measures overall efficiency of interns
* **Top Performer %** – Percentage of interns categorized as top performers

---

## 🛠 Tools Used

* Power BI (Dashboard development and visualization)
* DAX (for KPI calculations)

---

## 📊 Dashboard Features

The interactive dashboard includes the following components:

### 🔹 KPI Cards

* Displays key metrics such as total interns, average performance, and total tasks

### 🔹 Visualizations

* Department vs Performance (Bar Chart)
* Attendance vs Performance (Scatter Plot)
* Training Hours vs Performance (Column Chart)
* Productivity vs Performance (Scatter Plot)
* Performance Category Distribution (Pie Chart)

### 🔹 Filters (Slicers)

* Department
* Education
* Date
* Performance Category

---

## 🎨 Dashboard Design

* Clean and structured layout for better readability
* Interactive visuals enabling user-driven exploration
* Consistent color theme for clarity and professional presentation

---

## 🔍 Insights Enabled

The dashboard allows users to:

* Analyze performance trends across departments
* Understand the impact of attendance and training on performance
* Identify high-performing interns
* Explore productivity and task completion patterns
