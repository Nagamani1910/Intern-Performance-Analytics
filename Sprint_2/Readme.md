# 📊 Sprint 2 – Exploratory Data Analysis (EDA)

## 🎯 Objective

The objective of this sprint is to analyze the cleaned intern performance dataset and identify key trends, patterns, and relationships using exploratory data analysis techniques.

---

## 📁 Dataset Overview

The dataset contains detailed information about intern performance, including:

* Personal details: Name, Gender, Education, Department
* Work metrics: Tasks Completed, Productivity Ratio
* Training details: Training Hours per Day and Week
* Performance metrics: Performance Rate, Performance Category
* Attendance: Attendance Percentage, Attendance Category
* Feedback: Feedback, Feedback Score, Top Performer

---

## 🔄 Data Transformation & Feature Engineering

Data transformation was performed using Power Query to enhance the dataset and improve analysis:

* Cleaned and structured raw data
* Renamed columns for better readability
* Created new calculated columns for deeper analysis
* Derived features such as productivity ratio, performance category, and attendance category
* Generated additional tables to support better data modeling

These steps ensured high-quality data and enabled more meaningful insights.

---

## 🛠 Tools Used

* Power BI (Data visualization and analysis)
* Microsoft Excel (Data preparation)
* Power Query (Data cleaning and transformation)
* DAX (for creating calculated measures)

---

## 📐 DAX Measures Created

The following DAX measures were implemented to support analysis:

* **Avg Performance** – Average performance rate of interns
* **Avg Attendance** – Average attendance percentage
* **Avg Productivity** – Average productivity ratio
* **Avg Tasks Per Intern** – Average number of tasks completed
* **Total Interns** – Total count of interns
* **Total Tasks Completed** – Total tasks completed by all interns
* **Top Performer %** – Percentage of top-performing interns

---

## 📊 Analysis Performed

* Statistical analysis (average, total, distribution)
* Trend analysis using time-based data
* Pattern identification across variables
* Department-wise and category-wise comparison

---

## 📈 Visualizations Created

* Attendance vs Performance (Scatter Plot)
* Department vs Performance (Bar Chart)
* Training Hours vs Performance (Column Chart)
* Productivity vs Performance (Scatter Plot)
* Performance Category Distribution (Pie Chart)

---

## 🔍 Key Insights

* Higher attendance is associated with better performance.
* Increased training hours contribute positively to performance.
* Productivity ratio has a strong positive relationship with performance.
* Certain departments show higher performance and task completion.
* Top performers contribute significantly to overall productivity.
* Feedback scores align closely with performance outcomes.

---

## 📌 Conclusion

The exploratory analysis highlights that attendance, training, and productivity are key drivers of intern performance. These findings form the foundation for building an interactive dashboard in the next sprint.
