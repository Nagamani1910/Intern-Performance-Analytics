**Intern Performance Analytics and Insights Dashboard**

**Project Overview**
This project focuses on analyzing intern performance using data analytics techniques. It includes data collection, cleaning, transformation, and visualization to derive meaningful insights.

**Sprint 1 – Data Collection & Preparation**
In this phase, data was collected using Google Forms and enhanced with external datasets from Kaggle. The dataset was cleaned and transformed using Power BI (Power Query), and additional features were created to improve analysis.

**Dataset Details**
The dataset includes:
- Intern details (Name, Gender, Department, Education)
- Task completion and work hours
- Attendance percentage
- Performance rate
- Feedback and sentiment

**Data Processing Steps**
- Data collection using Google Forms  
- Data standardization and cleaning  
- Feature engineering using calculated columns  
- Data validation and transformation in Power BI
